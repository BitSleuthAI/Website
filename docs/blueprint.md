# **App Name**: BitSleuth Landing

## Core Features:

- Hero Section: Visually striking hero section with clear value proposition for BitSleuth at https://app.bitsleuth.ai/
- Features Showcase: Showcase of core features and benefits with interactive elements.
- Call-to-Action (CTA): Clear call-to-action (CTA) section encouraging user sign-ups.
- AI-Powered Copy Generation: Generate personalized marketing copy variants for the landing page using an AI tool to improve conversion rates.
- Contact Form: Contact form integration with simple email sending (no database).

## Style Guidelines:

- Primary color: Vibrant purple (#BF5FFF), derived from the dark mode primary color in the globals.css file, to convey innovation.
- Background color: Light gray (#F0F0F0), derived from the light mode background color in globals.css to ensure a clean, modern backdrop.
- Accent color: Blue-ish purple (#946BFF), analogous to the primary purple, used for highlights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif to maintain a clean, objective aesthetic for readability and consistency with the codebase
- Modern, minimalist layout with clear section divisions to improve readability.
- Clean and professional icons that complement the modern aesthetic, in the purple hue
- Subtle animations on scrolling and hover effects to engage users.
